# Portfólio Acadêmico - Algoritmos e Programação

Este repositório contém o portfólio de atividades práticas e projetos desenvolvidos para a disciplina de **Algoritmos e Programação** no curso de Engenharia de Telecomunicações.

## 🚀 Funcionalidades Principais
- **Vitrine Acadêmica:** Exibição estruturada de códigos-fonte, resoluções de problemas e documentação em PDF.
- **Visual Clean e Minimalista:** Interface inspirada em ambientes de desenvolvimento e terminais, focada em legibilidade.
- **Design Responsivo:** Adaptado perfeitamente para visualização em desktops e dispositivos móveis.

## 🛠️ Tecnologias e Ferramentas Utilizadas
- **Python:** Lógica fundamental, scripts automatizados e Programação Orientada a Objetos (POO).
- **HTML5 & CSS3:** Estruturação semântica e estilização customizada através de variáveis CSS.
- **Git & GitHub Pages:** Versionamento de código e hospedagem estática direta do portfólio.
- **Assistentes de IA (Claude & ChatGPT):** Ferramentas de apoio conceitual e pareamento de código (*code pairing*).

## 📂 Como Utilizar
Os arquivos deste repositório rodam nativamente no GitHub Pages. 
- O arquivo principal do site é o `index.html`.
- Para adicionar novos projetos, basta editar a seção `<div class="projects-grid">` dentro do arquivo `index.html` adicionando um novo bloco de `<article>`.

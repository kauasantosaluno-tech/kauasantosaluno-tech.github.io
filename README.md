# Portfólio Acadêmico - Algoritmos e Programação

Este repositório contém o portfólio de atividades práticas e projetos desenvolvidos para a disciplina de **Algoritmos e Programação** no curso de Engenharia de Telecomunicações por **Kauã Lopes**.

## 🚀 Funcionalidades Principais
- **Vitrine Acadêmica:** Exibição estruturada de códigos-fonte, resoluções de problemas e documentação em PDF.
- **Visual Clean e Minimalista:** Interface inspirada em ambientes de desenvolvimento, limpa e focada em legibilidade.
- **Design Responsivo:** Adaptado perfeitamente para visualização em desktops e dispositivos móveis.

## 🛠️ Ferramentas Utilizadas
- **ChatGPT:** Suporte conceitual e pareamento de código.
- **Gemini:** Auxílio no desenvolvimento lógico dos algoritmos.
- **Claude IA:** Refinamento técnico e revisão de estruturas de código.

## 📂 Como Utilizar
Os arquivos deste repositório rodam nativamente no GitHub Pages. 
- O arquivo principal do site é o `index.html`.
- Para adicionar novos projetos, basta editar a seção `<div class="projects-grid">` dentro do arquivo `index.html`.

Como publicar um arquivo

Fluxo de publicação:
1. Selecione o arquivo abaixo e preencha o título.
2. Clique em "Preparar publicação" — o sistema gera um arquivo posts.json atualizado para download.
3. Faça o commit de dois arquivos no seu repositório GitHub:
   • O arquivo original em /uploads/
   • O novo posts.json na raiz
4. Pronto! O site exibe a nova publicação automaticamente.

Nenhuma publicação cadastrada.

Após baixar, faça o commit deste arquivo na raiz do repositório (posts.json).

Como fazer o commit no GitHub

  1. Acesse github.com e abra seu repositório.
  2. Na aba Code, entre na pasta uploads/.
  3. Clique em Add file → Upload files e envie o arquivo.
  4. Volte à raiz do repositório, clique no arquivo posts.json e depois no lápis (editar).
  5. Selecione todo o conteúdo e cole o conteúdo do posts.json baixado aqui.
  6. Clique em Commit changes. O site atualiza em segundos!
